# CRUD de Produtos

Sistema web para cadastro, login de usuários e gerenciamento de produtos.

## 🛠 Tecnologias usadas

- **Frontend:** Vue.js
- **Backend:** Node.js + Express + Sequelize


## ▶️ Como executar

### Backend:
node server.js

### Frontend:
npm run dev

### Funcionalidades
--> Cadastro e login de usuários

--> Listagem de produtos

--> Criação, edição e exclusão de produtos
